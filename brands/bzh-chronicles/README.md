# BZH Chronicles

Direction pour les icônes de la famille BZH Chronicles.

## Principes

- fond sombre ;
- contraste fort ;
- symbole central très simple ;
- touches cyan, magenta ou rouge ;
- lisible en très petit format.

## Symboles possibles

- triskel ;
- carte ;
- étoile ;
- faille ;
- signal ;
- arbre sacré.
