# Convention Sterenna Icons

## Nom du repo

```txt
Sterenna-studio/sterenna-icons
```

## Identifiant d'icône

Chaque projet possède un `iconId` stable.

```txt
<scope>-<project>
```

Exemples :

```txt
star-gwen-ha-star
bzh-universe
nitro-clicker
botanica-obscura
nitro-skill-arena
nitro-titan-rocket-run
pokegang
chronicles-tcg
bzh-breach-storm
bzh-nemeton-lockdown
spirit-overdrive
```

## Kit minimum par repo web

```txt
favicon.ico
favicon.svg
favicon-32.png
apple-touch-icon.png
site.webmanifest
```

## Arborescence par icône

```txt
sources/<iconId>/icon.svg
sources/<iconId>/icon-dark.svg
sources/<iconId>/icon-light.svg
sources/<iconId>/notes.md
exports/<iconId>/favicon.ico
exports/<iconId>/favicon.svg
exports/<iconId>/favicon-32.png
exports/<iconId>/apple-touch-icon.png
exports/<iconId>/site.webmanifest
```
